# The Hermetic Codex

A modern, mobile-friendly guide to Hermetic philosophy and the ancient wisdom of Hermes Trismegistus.

## Features

- **The 7 Hermetic Principles** - Interactive cards explaining each universal law
- **Mental Transmutation Tool** - Visualize polarity and emotional transmutation
- **The Emerald Tablet** - Cryptic wisdom with logical interpretations
- **Self-Assessment Radar** - Track your alchemical balance across all principles
